Dicas de Cuidados com a Pele

1. Sempre limpe o rosto antes de dormir para remover maquiagem e impurezas.

2. Use um hidratante adequado para o seu tipo de pele.

3. Aplique protetor solar diariamente para proteger a pele dos raios UV.

4. Exfolie regularmente para remover células mortas e promover a renovação celular.

Dicas de Maquiagem

1. Comece com um primer para uma base suave e maquiagem de longa duração.

2. Use uma base que combine com o seu tom de pele para um look natural.

3. Misture bem a sombra para um acabamento suave.

4. Não esqueça de finalizar a maquiagem com um spray fixador para durabilidade.
